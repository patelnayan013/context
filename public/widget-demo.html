Quick Start

Add this script tag to any website to embed the widget:

<!-- Add before closing </body> tag --> <script src="https://your-domain.com/widget.js" data-api-url="https://your-domain.com" data-position="bottom-right" data-theme="light" data-primary-color="#6366f1" data-title="Ask us anything" defer ></script>

Configuration Options

API URL

data-api-url

Your API endpoint URL that handles chat requests

Position

data-position

Widget position: bottom-right, bottom-left, top-right, top-left

Theme

data-theme

Color theme: light or dark

Primary Color

data-primary-color

Hex color code for branding (e.g., #6366f1)

Title

data-title

Header text displayed in the chat panel

Placeholder

data-placeholder

Input field placeholder text

Welcome Message

data-welcome-message

Initial greeting message from the assistant

JavaScript API

// Programmatically control the widget window.KnowledgeWidget.open(); // Open the chat panel window.KnowledgeWidget.close(); // Close the chat panel
